# TinySpiderServer
# 简易爬虫服务器

**开发环境：Ubuntu**

**主要内容：编写小型服务器，并且利用此服务器爬虫抓取科大最新的招聘信息**

**技术难点：用C语言进行网络套接字编程，编写多线程版服务器，http协议报文的解析，C++爬虫抓取网页，正则表达式匹配关键信息。**

**经历/收获：学习了套接字编程和http协议，了解爬虫，学习使用libcurl库爬虫网页，并基于正则表达式匹配所需信息。**
